# Eagle Lion System Technologies - Aggregator Dashboard

## Project Info

This is the Enterprise Transaction Aggregator Dashboard for Eagle Lion System Technologies LTD. It allows managing transactions, tracking payments, and monitoring providers.

## Getting Started

To run the project locally, follow these steps:

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository

   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory

   ```sh
   cd aggre-dash-main
   ```

3. Install dependencies

   ```sh
   npm install
   ```

4. Start the development server
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the project for production:

```sh
npm run build
```
